## Overview

Magento 2 extension for integrating AirRobe's circular wardrobe functionality into Magento storefronts.

## Tech Stack

- PHP 8.2/8.3, Magento 2.4.6+/2.4.7+
- Docker (markshust/docker-magento for dev)
- Composer

## Commands

- **Install (Composer):** `composer require airrobe/magento-extension`
- **Install (zip):** Extract to `app/code/AirRobe/` directory
- **Dev environment:** Docker-based via markshust/docker-magento

- Configuration managed via Connector Dashboard
